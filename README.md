# WeatherApp

### Overview

The purpose of this project was to create an asynchronous web application that uses an external web API, POST and GET endpoints with a node.js/express based server that allows to automatically update the UI based on userinput.

### Instructions to run the application

1. `git clone https://github.com/JZerman2018/my-weather-journal-app.git`
2. Install the depencencies:

- `npm install express`
- `npm install cors`
- `npm install body-parser`

3. Run `npm server.js` to start the app.
